日志分析

evtx文件，可以直接使用windows分析，也可以用第三方工具,日志文件中得到密码,就是很常见的误操作场景，把密码当用户名输入了，被记录在日志中：

  • omerbenamram/evtx: A Fast (and safe) parser for the Windows XML Event Log (EVTX) format
    https://github.com/omerbenamram/evtx
