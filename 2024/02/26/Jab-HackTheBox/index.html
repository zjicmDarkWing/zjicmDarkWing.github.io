端口扫描

除了常规域端口之外，可以看到xmpp相关端口：

XMPP

搜索可以知道XMPP是一种IM协议，可以使用任意客户端连接：

  • XMPP | The universal messaging standard
    https://xmpp.org/
  • Pidgin :: Pidgin, the universal chat client
    https://www.pidgin.im/
  • Adium - Download
    https://adium.im/

然后查看聊天室列表得到一个新域名，同样加hosts：

users

然后在用户目录中得到很多用户名(两千多个):

ASREPRoast

现在我们只有大量用户名，那首先尝试ASREPRoast：

  • ASREPRoast - HackTricks
    https://book.hacktricks.xyz/windows-hardening/active-directory-methodology/asreproast

得到几条hash，可以破解出jmontgomery的密码

jmontgomery

然后使用得到的jmontgomery账号密码登录xmpp，发现一个新的聊天室：

pentest2003

加入这个聊天室，其中得到svc_openfire密码：

user flag

现在openfire可以收集bloodhound进行分析：

可以发现svc_openfire在Distributed COM Users组中

那就可以直接dcomexec：

  • impacket dcomexec | WADComs
    https://wadcoms.github.io/wadcoms/Impacket-DCOMExec/

打到svc_openfire shell，桌面得到user flag：
