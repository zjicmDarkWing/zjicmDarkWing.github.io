基本信息

  • https://www.hackthebox.com/home/machines/profile/474
  • 10.10.11.158

端口扫描

windows 域机器：

80

直接访问是IIS默认页面：

443 watch.streamIO.htb

证书里可以得到域名，加hosts后访问：

443 streamIO.htb

streamIO.htb

sql注入

login存在注入,很费时间，最终得到hash，破解出来密码：

admin

admin中本身有几个功能,fuzz参数可以发现debug：

debug

debug那里可以使用php伪协议：

master

继续进行fuzz可以发现master.php,发现其中调用eval,根据代码分析构造请求，RCE:

master.php

reverse shell

可以直接msf生成上线命令，方便后续操作：

portfwd 1433

前面已经得到数据库账号密码，转发端口：

backup

数据库可以发现streamio_backup,users表中可以获得nikk37用户密码，这是机器上有的账户：

得到的hash直接在线解密，得到明文密码：

user flag

得到的账号密码登录，user flag

需要把stream.htb也加到hosts

提权信息

firefox

机器上装有firefox，提取登录信息进行解密

  • lclevy/firepwd: firepwd.py, an open source tool to decrypt Mozilla protected passwords
    https://github.com/lclevy/firepwd

bloodhound

bloodhound收集信息，发现可以从JDgodd打到Administrator

提权

core staff

根据bloodhound的信息，JDgodd对core staff有WriteOwner权限，core staff有权限去获取laps密码，所以首先把JDgodd加到core staff组中：

laps password

现在可以使用JDgodd去获取laps密码：

root flag

得到的laps密码就是Administrator密码,flag在Martin桌面：

hash
